CFX Asset Escrow

The Asset Escrow system is a ownership system created by the Cfx to protect both customers and sellers.

You asset (scripts, maps, etc) are delivered automatically by the system and will appear in the keymaster under the Granted Assets tab.

Common issues​

Error: syntax error near '<\​1>'​

This error indicates that your package has been corrupted or that your server artifacts ared outdated.

If your server is outdated please update it to the latest recommended build, wich you can find here.

If thats not the case, then your package is corrupted. Often, this is caused by FTP softwares like FileZilla, while transferring the files to your server.

To fix this, you have to either use WinSCP, wich is a different FTP software or set FileZilla to binary mode.
